# Frontend (MesuKoros)

React + Vite app for MesuKoros.

## Local run

```bash
npm install
npm run dev
```

By default this app calls `/api`.
For separate backend hosting set `VITE_API_URL`.

## Environment
Copy `.env.example` to `.env` and set:

- `VITE_API_URL=https://your-backend-domain.com/api`

## Build

```bash
npm run build
npm run preview
```
